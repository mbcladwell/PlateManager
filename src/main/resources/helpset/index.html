LIMS*Nucleus

Introduction

LIMS*Nucleus is an application for managing multi-well plates containing samples. Samples have assaociated assay data which can be used to identify assay hits. Hits can then be rearrayed into new plates

LIMS*Nucleus utilizes the following entities:

  • Project
  • Plate Set
  • Plate
  • Well
  • Sample
  • Assay Type
  • Assay Results
  • Hit list
